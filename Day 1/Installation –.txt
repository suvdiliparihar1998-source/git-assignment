1. Installation – Basic Tools

1.1 Install Git


Go to git-scm.com


Download Git for your OS (Windows/Mac/Linux)


Run the installer


Keep default options → click Next till Finish


Verify installation:
git --version




1.2 Install VS Code


Go to code.visualstudio.com


Download VS Code


Install with default settings


Open VS Code


(Optional but recommended)


Install extensions:


GitLens


ESLint


Prettier


MySQL / SQL tools







1.3 Install Node.js


Go to nodejs.org


Download LTS version


Install → keep all defaults


Verify installation:
node -v
npm -v
